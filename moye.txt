1. What is the primary purpose of a stack in data structures?
   - A) First-In-First-Out (FIFO)
   - B) Last-In-First-Out (LIFO)
   - C) Random access

2. Which data structure is best suited for quickly finding the smallest element in a collection?
   - A) Queue
   - B) Heap
   - C) Linked List

3. In which scenario is a hash table most efficient?
   - A) Storing sorted data
   - B) Searching for an element by index
   - C) Rapid data retrieval by a known key

4. What is the time complexity for searching an element in a balanced binary search tree?
   - A) O(1)
   - B) O(log n)
   - C) O(n)

5. Which data structure is used for graph traversal algorithms?
   - A) Stack
   - B) Queue
   - C) Array

6. What is the primary advantage of using a linked list over an array?
   - A) Constant time access to any element
   - B) Efficient memory utilization
   - C) Ability to resize dynamically

7. Which sorting algorithm has an average-case time complexity of O(n log n)?
   - A) Bubble Sort
   - B) Merge Sort
   - C) Insertion Sort

8. What is the main purpose of a priority queue?
   - A) Random access to elements
   - B) Efficiently finding the maximum element
   - C) Maintaining a first-in-first-out order

9. Which operation takes constant time in a hash table with a well-designed hash function?
   - A) Insertion
   - B) Deletion
   - C) Searching

10. What is the primary drawback of using a linear search algorithm?
    - A) Inefficient for large datasets
    - B) Requires a sorted collection
    - C) Consumes excessive memory


Here are the answers:

1. B) Last-In-First-Out (LIFO)
2. B) Heap
3. C) Rapid data retrieval by a known key
4. B) O(log n)
5. B) Queue
6. C) Ability to resize dynamically
7. B) Merge Sort
8. B) Efficiently finding the maximum element
9. A) Insertion
10.A) Inefficient for large datasets
